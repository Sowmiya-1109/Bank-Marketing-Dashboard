Bank Marketing Dashboard

Project Overview

The Bank Marketing Dashboard is an interactive Power BI project that analyzes customer and marketing campaign data. The dashboard helps identify customer behavior, deposit subscription patterns, balance distribution, and campaign performance to support better business decisions.

Tools Used

- Microsoft Power BI
- Microsoft Excel

Dashboard Pages

1. Dashboard

Provides an overview of key performance indicators (KPIs), customer demographics, deposit statistics, and interactive filters.

2. Customer Analysis

Analyzes customer age, balance, loan status, and marital status to understand customer characteristics.

3. Deposit Analysis

Explores deposit subscriptions across age groups, job categories, and marital status.

4. Balance Analysis

Examines customer balance groups and their impact on deposit subscriptions.

5. Duration Analysis

Analyzes the relationship between call duration and successful deposit subscriptions.

6. Key Insights & Recommendations

Summarizes the major findings and provides business recommendations based on the analysis.

Features

- Interactive slicers
- KPI Cards
- Bar Charts
- Column Charts
- Pie Charts
- Scatter Charts
- Business Insights & Recommendations

Key Insights

- Total Customers: 11,162
- Total Balance: 17M
- Average Balance: 1.53K
- Deposit Subscription Rate: 47.38%
- Customers with higher balances show better deposit subscription rates.
- Longer call durations are associated with higher deposit subscriptions.

Dataset

This project uses the **Bank Marketing Dataset** obtained from Kaggle for educational and portfolio purposes.
